## Judwa Approach (Multiplication Tricks)

- **ab + ba**: Multiply by 11
- **ab − ba**: Multiply by 9
- **Multiplication by 11**: `XY × 11 = X(X+Y)Y`

## Algebraic Identities

### Basic Formulas

1. $a^2 - b^2 = (a+b)(a-b)$
2. $(a+b)^2 = a^2 + b^2 + 2ab$
3. $(a-b)^2 = a^2 + b^2 - 2ab$
4. $(a+b)^3 = a^3 + b^3 + 3ab(a+b) = a^3 + b^3 + 3a^2b + 3ab^2$
5. $(a-b)^3 = a^3 - b^3 - 3ab(a-b) = a^3 - b^3 - 3a^2b + 3ab^2$
6. $a^3 - b^3 = (a-b)(a^2 + b^2 + ab)$
7. $a^3 + b^3 = (a+b)(a^2 + b^2 - ab)$
8. $(a+b+c)^3 = a^3 + b^3 + c^3 + 3(a+b)(b+c)(c+a)$
9. $(a+b+c)^2 = a^2 + b^2 + c^2 + 2ab + 2bc + 2ca$

## Laws of Exponents

| Law | Formula |
|-----|---------|
| I   | $a^n \times a^m = a^{m+n}$ |
| II  | $\dfrac{a^n}{a^n} = a^{m-n}$ |
| III | $(a^n)^m = a^{mn}$ |
| IV  | $(ab)^n = a^n \times b^n$ |
| V   | $\left(\dfrac{a}{b}\right)^n = \dfrac{a^n}{b^n}$ |
| VI  | $a^0 = 1 \quad (a \neq 0)$ |
| VII | $a^{-n} = \dfrac{1}{a^n}$ |
